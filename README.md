# Text preprocessing

The first steps of any NLP pipeline, done by hand in one script fora text mining course.

## What it covers

- tokenization three ways: plain Python split, NLTK and spaCy
- sentence tokenization
- stopword removal
- stemming with Porter
- lemmatisation with WordNet and spaCy
- small exercises on real, messy sentences with contractions and abbreviations

## How to run

You need Python 3 with nltk and spacy installed. The install and download commands are commented at the top of the file. Then just run it and read the printed output next to each exercise.
